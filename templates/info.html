Data Analysis Projects :-

The process of analyzing the data and getting valuable insights from it in Graphical or Numerical Format by cleaning and applying statistical techniques is called Data Analysis.Data Analysis consists of various steps and to properly analyze the data these set of steps should be followed which will ultimately result in Valuable Insights.Data Understanding , Feature Engineering, Feature Selection, Plotting graphs , performing Statistical Analysis like outlier detection are some of the most important steps involved in the Data Analysis Project or say Data Analysis Life Cycle.Python libraries like Pandas, Numpy, Sci-Py , Pandas-Profiling are used for performing Data Analysis. For plotting the data and creating Visualizations we can make use of Libraries like Matplotlib and Seaborn. All in all there are various libraries involved in building successful Data Analysis Life Cycle!
